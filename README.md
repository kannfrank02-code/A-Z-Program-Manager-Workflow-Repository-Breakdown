# A–Z Program Manager Workflow Repository

This repository is a complete operating system for managing programs from intake through closeout.

It is designed for Program Managers, Project Managers, Operations Coordinators, Executive Assistants, PMO Analysts, and administrative professionals who support complex initiatives.

## Purpose

The purpose of this repository is to provide repeatable tools for organizing people, timelines, deliverables, risks, budgets, meetings, decisions, communication, training, adoption, and closeout.

## How to Use This Repository

Start with the Program Intake Form. Then move through chartering, planning, stakeholder mapping, execution, reporting, risk management, training, and closeout.

## Program Manager Philosophy

A strong Program Manager makes complex work visible, organized, measurable, and actionable.

The goal is not to do everyone’s job. The goal is to create the structure that helps everyone do their job on time, with clarity and accountability.
