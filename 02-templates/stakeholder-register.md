# Stakeholder Register

Use this to track everyone involved in the program.

## Instructions

Put every person or group connected to the program here.

Include:
- Sponsors
- Leaders
- Workstream owners
- Finance
- HR
- IT
- Vendors
- End users
- Anyone impacted by the program

|  | Workstream Lead |  | Medium | High | Tasks, deadlines, blockers | Weekly | Program Manager |
|  | End User |  | Low | High | Training, process changes, launch date | As Needed | Change Lead |
