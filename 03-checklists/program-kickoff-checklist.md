# Program Kickoff Checklist
Use this checklist before, during, and after the kickoff meeting to make sure the program starts with clear ownership, scope, timeline, risks, communication, and next steps.
## Before the Kickoff Meeting

Complete these steps before the first official program meeting.

- [ ] Confirm the executive sponsor
- [ ] Confirm the business owner
- [ ] Confirm the program manager
- [ ] Complete the program intake form
- [ ] Draft the program charter
- [ ] Identify key stakeholders
- [ ] Create the stakeholder register
- [ ] Create the initial project plan
- [ ] Identify known risks and dependencies
- [ ] Create the shared folder or documentation location
- [ ] Create the program tracker
- [ ] Schedule the kickoff meeting
- [ ] Send the kickoff meeting invitation
- [ ] Send the kickoff meeting agenda
- [ ] ## During the Kickoff Meeting

Use the meeting to align everyone on the purpose, scope, roles, timeline, risks, and communication plan.

- [ ] Review the program purpose
- [ ] Review the business need
- [ ] Review the goals and success measures
- [ ] Review what is in scope
- [ ] Review what is out of scope
- [ ] Review the timeline and major milestones
- [ ] Confirm roles and responsibilities
- [ ] Confirm decision makers
- [ ] Review known risks
- [ ] Review known dependencies
- [ ] Confirm the communication plan
- [ ] Confirm the reporting cadence
- [ ] Confirm the meeting cadence
- [ ] Capture questions
- [ ] Capture decisions
- [ ] Capture action items
## After the Kickoff Meeting

Complete these steps within 24 hours after the kickoff meeting.

- [ ] Send a kickoff meeting recap email
- [ ] Share meeting notes
- [ ] Update the project plan
- [ ] Update the RAID log
- [ ] Update the stakeholder register
- [ ] Add new risks or dependencies
- [ ] Add decisions to the decision log
- [ ] Assign action items with owners and due dates
- [ ] Save all documents in the shared folder
- [ ] Confirm the next meeting date
- [ ] Follow up on unanswered questions
- [ ] Send any materials promised during the meeting

## Kickoff Is Complete When

The kickoff process is complete when:

- [ ] The sponsor is confirmed
- [ ] The business owner is confirmed
- [ ] The program manager is confirmed
- [ ] The program charter is drafted or approved
- [ ] Stakeholders are identified
- [ ] The initial timeline is created
- [ ] Risks and dependencies are documented
- [ ] Meeting cadence is confirmed
- [ ] Reporting cadence is confirmed
- [ ] Action items have owners and due dates
- [ ] Meeting recap has been sent
- [ ] All documents are saved in the shared location
